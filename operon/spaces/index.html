Hugging Face Spaces

Interactive front doors into the structure without installing the code first.

Epistemic Topology Explorer

operon-epistemic

Use this to explore topology classes, architecture advice, and practical pattern recommendations.

Diagram Builder

operon-diagram-builder

Use this to start from practical presets like reviewer gate or specialist swarm, then inspect the generated structure.

Bi-Temporal Memory Explorer

operon-bitemporal

Use this to explore bi-temporal fact management with dual time axes, point-in-time queries that reconstruct belief state at any historical coordinate, and temporal diffs showing how corrections preserve full history.

Watcher Dashboard

operon-watcher

Use this to explore three-category signal classification (epistemic, somatic, species-specific), intervention timelines showing retry/escalate/halt decisions, and configurable thresholds.

Adaptive Assembly

operon-adaptive

Use this to explore task fingerprinting and template scoring from the PatternLibrary, adaptive organism assembly from ranked templates, and the experience pool that drives intervention recommendations.

Sleep Consolidation

operon-consolidation

Use this to explore the sleep consolidation cycle: replay successful patterns into episodic memory, compress recurring patterns into templates, run counterfactual analysis over corrected bi-temporal facts, and promote histone marks from temporary to permanent.

How These Fit

The current demos are best understood as interactive front doors into the same structure the library uses locally. They are not separate products.
