Where to start in the example catalog without reading everything in numerical order.

Operon now has many examples. New users should not start from the beginning and read them in order.

Recommended Entry Path

  • examples/68_pattern_first_api.py
  • examples/69_skill_organism_runtime.py
  • examples/67_epistemic_topology.py

This order is deliberate:

  • 68 gives you the shortest path into the public wrappers
  • 69 shows the practical multi-model runtime
  • 67 shows the epistemic analysis layer underneath
