Operon: Roadmap Complete

Eight Phases, Six Layers, 1130 Tests, and What It Means to Build Agents That Grow Up

Bogdan Banu · March 2026 · github.com/coredipper/operon

Abstract

The eight-phase roadmap that began with bi-temporal memory in v0.19 is now complete. Operon has grown from a library of typed wiring patterns into a cognitive architecture where agents have auditable memory, adaptive structure selection, sleep consolidation, social learning, curiosity, and developmental staging with critical periods. This post recaps the arc, the numbers, and what comes next.

3. What the Paper Now Says

The article abstract has been rewritten. It no longer describes just structural safety patterns. It describes a six-layer progression from typed interfaces through temporal epistemics, adaptive structure, cognitive architecture, and developmental staging. The conclusion traces the arc from v0.17 through v0.23 and identifies the remaining open directions: convergence with operational runtimes (AnimaWorks, Swarms), production benchmarks (BFCL, AgentDojo), adversarial robustness, and learned diagram optimization.

4. The Integration Tests

Phase 8’s five integration tests prove the full stack works:

  1. Organism with bi-temporal substrate and watcher — facts recorded, watcher observes, belief state reconstructible
  2. Adaptive assembly followed by sleep consolidation — run → consolidate → promote templates
  3. Social learning with developmental gating — mature teacher scaffolds young learner, critical period closes
  4. Full lifecycle — telomere → development → library → adaptive → consolidation
  5. Memory adapters — histone and episodic memories bridged into bi-temporal fact store

Why Integration Tests Matter Here

Each phase was tested in isolation. The integration tests prove that the subsystems compose: that a watcher can observe a substrate-equipped organism, that consolidation can distill adaptive runs, that scaffolding respects developmental gates. The individual unit tests prove correctness; the integration tests prove coherence.

5. What Comes Next

The roadmap is complete. The next directions are:

  • Convergence investigation — combining Operon’s structural guarantees with AnimaWorks’ cognitive runtime and Swarms’ enterprise orchestration (plan doc)
  • Production benchmarks — BFCL and AgentDojo evaluation at scale
  • Paper submission — the article is publication-ready with the v0.23.1 polish

The deeper point remains the one from the first blog post: agents that cannot explain their past decisions are agents you cannot trust with consequential ones. The roadmap built the explanation layer (bi-temporal memory), the adaptation layer (pattern library + watcher), the cognitive layer (consolidation + social learning), and the developmental layer (critical periods + scaffolding). What started as typed wiring diagrams now has temporal epistemics, dreams, curiosity, and the ability to grow up.
